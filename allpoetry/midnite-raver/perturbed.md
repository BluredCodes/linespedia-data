if i could just,
spin, spin, run, run,
if i could just,
run, and, run, and,
if i could just,
spin, and, spin, then,
i would all the time,

if i could just,
go, go, drown, drown,
if i could just,
drown, and, drown, and,
if i could just,
go drown, go drown,
i would all the time,

one more day to,
fall, fall, down, down,
one more day to,
fall, and, fall, and
one more day to,
fall down, fall down,
before i find myself.
