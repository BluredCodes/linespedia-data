---
title: "what i need"
writer: "midnite raver"
slug: "what-i-need"
source: "AllPoetry"

more time for me,
more time to breathe,
more time to see,
more time to think,
more time to fall,
more time to cry,
more time to feel,
more time to die,
more time to dull,
more time to waste,
more time to try,
more time to fly,
more time to rest,
more time to sleep,
more time to dream,
more time to bleed,
more time to wait,
more time to kill,
more time for fun,
more time for sun,
more time alone,
more time inside,
more time to hide,
more time for slides,
more time to swim,
more time to dive,
more time to drown,
more time alive.
