I can be sexy
I can be sweet
I can be cool
I can be neat
I can be honest
I can be true
I can be happy
I can be blue
I can be funny
I can be mad
I can be stupid
I can be sad
I can be anything I want to be
I can be something that you can't see
I can be anything, and it's true
I'm just happy, that I'm not youhehe ppl make me mad and im glad i'm ME and not them
