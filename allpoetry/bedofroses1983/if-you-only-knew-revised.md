title: "If You Only Knew (Revised)"

If you only knew the number of nights I've laid awake thinking of you,
the number of tears I have shed over you.
If you only knew the number of times I nearly told you how I felt about you,
the number of times I've wanted to kiss you.
If you only knew....If you only knew.
But you don't know and now it is too late. You have married and moved away.
It is too late...
If you only knew that I love you.
