Now, now, now...
Which Now is Now?
Will the Real Now Please stand?

And, when is it 'Now,' exactly?
How long does it last?
How fast does it pass?
Is it longer/shorter than
a nano/atto/pico...second?

If no unit of Time ever rests or lasts;
and if each 'Now' is infinitesimally divisible--
Then is 'Now', in fact, never?
Or, is 'Now' always...Now-ing?

--Uncle Greg
