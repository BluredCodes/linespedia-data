Invible.
None can see me.
No one wants to see me.
I used to have a reason for getting up every morning.
That feeling is fadied.
Like a phantom.
Hardly seen.
No words.
Never felt.
No life.
No dreams.
Only Tears.
Only Memories of days gone bye.your input is valuable.
