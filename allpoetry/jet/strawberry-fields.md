---
title: "Strawberry Fields"
writer: "Jet"
slug: "strawberry-fields"
source: "AllPoetry"
url: "https://allpoetry.com/poem/10908-Strawberry-Fields-by-Jet"
---

The strawberry fields in my mind
Stretch out to fields afar.
I wander in those sprawling fields
And wonder where you are.

I wonder if you're hurting still
Or did you stop the pain?
I wonder if you're no longer scared
Or is your fear always the same?

I wonder if you're lonely.
I wonder if you're lost.
I wonder if you'd still believe
That it was worth the cost.

The strawberry fields we shared
Have been silent since you've gone.
I wonder if you're watching now
If you'd want me to move on.

With strawberry blossoms in my hand
I'll try to start anew.
And I know one day I'll find myself
In strawberry fields with you.
