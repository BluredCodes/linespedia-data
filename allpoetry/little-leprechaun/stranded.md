'Stranded'   9 - 18 - 01

This pain is just so great
You're so alone
Blocked by this mindgate
You're so alone
Feelings of love into hate
You're so alone
Light doesn't penetrate
You're so alone....

Chorus
THIS PAIN IS SLAMMING!!
You're so alone
THIS KNIFE IS JABBING!!
You're so alone
THESE EYES ARE CRYING!!
You're so alone
THIS SOUL IS DYING!!
You're so alone....

This heart is striving
for the love it wants
This body is grabbing
for the life it needs
This soul is trying
for its forgiveness
This mind is prying
possible deliverance

(repeat chorus)

This will is defying
You're not alone
This heart is trying
You're not alone
This love is reviving
You're not alone
This body is climbing
You're not alone....

Little Leprechaun
