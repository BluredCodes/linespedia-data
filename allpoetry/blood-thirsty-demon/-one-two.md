~ } ~ } One....  Two....

~ ~ >> Fantastic relief hits in this
darkened Abyss, and now...
....I can honestly feel You.


~ } ~ } Three....  Four....

~ ~ >> An OutStanding Amount of
MidNight-Red BlooD streams, oh, so...
....Steadily to the floor.


~ } ~ } Five....  Six....

~ ~ >> Exhilerating physical pain
grimmly reveals such...
....UnEarthed moving tricks.


~ } ~ } Seven....  Eight....

~ ~ >> Solely from My BlooDieD wrists,
sliced ever so downwards...
....All too purposely straight.


~ } ~ } Nine....  Ten....

~ ~ >> Drifting off, passing-out, 
awaken in the ICU with SelfDisgust...
....OverWhelming again.
