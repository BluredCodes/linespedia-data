I wonder what it would be like if he never kissed me...
   Would you be here with me?
I wonder what it would be like if you weren't so upset with me...
   Would we even be friends?
I wonder what it would be like if you talked to me...
   Would you forgive me?
I wonder what it would be like if we got back together...
   Would you trust me?
I wonder what it would be like if I could stop crying over you...
   Would I miss you any less?
I wonder what it would be like if I moved on...
   Would I ever get you back in my life?
I wonder what it would be like if I could get over you...
   But I can't.
   It's too hard.
   I care about you too much.
   I just can't let go.
