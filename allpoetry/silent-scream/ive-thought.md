title: "I've thought"
writer: "silent scream"

I've thought about it 
I'm ready for it
who cares how old

I've thought about it
I'm waiting for it
so I've been told

I've thought about it
I'm holding around it
Until the end I know

I've thought about it
I'm living
letting colours grow

For knowing this
feels just like a kiss

Deep in the night
the moment's so right

I've thought about it
I'm here for it
I just want to let you know.
