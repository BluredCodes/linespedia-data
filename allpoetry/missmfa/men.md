Some are summer 
Some are rain 
Some are armour 
Some are pain 
Some are murmurs 
Some are stains 
Some are clamours 
Some are chains 
Some are stammers 
Some, refrains 
Some are farmers 
Some, the grains 
Some are tremours 
Some are plains 
Some are charmers 
Some are Cain 
Some are magma 
Hot terrain 
Some are Karma 
Pre-ordained.
