True beauty is not found in roses by an old Victorian house.
not in  gold
It is not rubies, sapphires, or even diamonds.

For although it is precious as gold
delicate as roses
and everlasting as a diamond,
True beauty 
is found only in love
only in peace.
True beauty
is found only in courage.
only in honesty.
True beauty can only be found within yourself...
