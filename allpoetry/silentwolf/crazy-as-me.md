Sometimes I sit and just turn on the music
As I let myself drift away into another place 
Where I can no longer control my sanity...
And so my journey goes..

A big green baboon yells at you from a tree
Wouldn't you like to be as crazy as me?

A giant pink elephant on the trapeze
Wouldn't you like to be as crazy as me?

The grandest flea circus minus the fleas
Wouldn't you like to be as crazy as me?

A grand five course meal, the only food peas
Wouldn't you like to be as crazy as me?

Take a shit in a toilet, then while flushing yell "FLY FREE!"
Wouldn't you like to be as crazy as me?

Wait till the ride stops, then yell out "WHEEEEEEEEE!!!!!"
Wouldn't you like to be as crazy as me?

Dub your best friend Monkey Chow Chow (MCC)
Wouldn't you like to be as crazy as me?

Pick up a penny, then shriek loudly with glee
Wouldn't you like to be as crazy as me?

Invite people to a party, under subject write "fighting with cheese"
Wouldn't you like to be as crazy as me?

Run into a wall, wake up next to a tree
Wouldn't you like to be as crazy as me?

In a crowd of people, pick something up, throw it, then flee
Wouldn't you like to be as crazy as me?

For a birthday present give someone copies of their own keys
Maybe you wouldn't want to be as crazy as:
Big
Small
Short
Tall
Boring
Fun
