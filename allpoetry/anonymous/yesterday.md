Where have we gone since yesterday
I thought we'd live to see forever
But forever has gone away
And now we live straight on till never
All the loss, all the hurt
Don't mean a thing anymore
Because we're in the dirt
Below the floor
If only we could make yesterday right
If only we could still do that today
A candle would burn to expose the night
And we'd never worry again about yesterday
