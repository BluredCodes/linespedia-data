Scram, scram, scram - Dead girl on the run 
Destiny calls you - as death appears, 
Rock and roll - as the devil comes 
to take you away - to another time - another place. 

No smile on your face just a voluptuous grin and lots of skin 
My precious one - my sweetest delight - put your head in a vice, 
Jump a little - hump a little - better to die - than twiddle 
you're stuck in the middle with no escape. 

Just a silly orgiastic grin on your face 
panties awry - pretty thighs - many sighs - from lots of guys, 
Mount the white pony tonight and flee Heavens rage 
Is there hope for the darkness in your mind? 

Chorus..... 

Dead girl on the run - run run run run run 
life's to short - it will eat your heart, 
So have some fun and run run run run run 
dead girl on the run - dead girl on the run 


Universe full of traitors - nothing righteous 
running like hell - dead but not knowing - everything showing 
all the blood flowing - so dead yet so precious 
Dead girl on the run ... Dead girl on the run. 

Tag....... 

Get her a rope 
Dead girl on the run 
Hang her high 
Dead girl on the run 
Let her swing 
Dead girl on the run.
