title: "'Ahhh so cute'"

"Ahhh so cute" as it may seem
An isz by either colour does seem to be
But no one wants to get close
They poke and tease
Then run away
Not to be chased
But to just follow something else
We run free in the outback
Mindless
Boneless
Teeth as sharp as daggers
But look so cute
Why am I the only one of my kind cursed so
I can feel
Pain
I have emotions
That are crushed
I have a mind
Riddled with this curse of mine
I have bones
That can be broken
I am set apart from them all
But yet others still run away
Are isz's just not ment to be happy
We can run free for all time
But I don't run for food like the others
I run to find my other half
That part of me that can let me ack like the rest
Happy in there outback
The isz's run free
In the city I am trapped
This isz can't run
This isz is not free
For I'm trapped
There is no love to set me free

By: Isz
