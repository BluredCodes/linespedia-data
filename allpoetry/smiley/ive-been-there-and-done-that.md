No matter what you do or say.

I'VE BEEN THERE AND DONE THAT! 

So you say that you still want
to play.

I'VE BEEN THERE AND DONE THAT!

Now the years has passed us by
and I'm still asking WHY?
Now I know that you're not
coming back, it doesn't matter
because,

I'VE BEEN THERE AND DONE THAT!

So, stay out there in the wilderness
PLEASE because there is no need to 
come back to me.

I'VE BEEN THERE AND DONE THAT!

Yvonne J. Griffin
