i want it alli want it all back.we used to laugh,we used to cry,together.rooms full of love.we were home.i want it alli want it all back.where did it gohave we grown aparti want the love.i want the security.i want it alli want it all back.there was a placethere was a homewe all could live.we all could grow.i want it all i want it all back.i feel lost.i'm all alone.where did you run towhere have you gonethe foundation's brokenthe house is wornour ties have come undonewe've gone our separate waysi want it all.i want it all back.pay special attention to the punctuation and the lack there of in this one.  i'll see if anyone catches what it may mean, if not, i'll explain it.  it is a bit out there.  but then again, so am i, and so is this situation...
