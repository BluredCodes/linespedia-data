title: "You Can't Be Me"

You can't be me, and if you try, you'll only be a hypocrite.I know this because I've tried, but couldn't do it.I want to know when to stop, I want to learn to how to say, 'no'.I want to know how much to give, and how much to let go.I want to know how to resolve issues, and I want things to be the same.But if that wish were granted - no one would know my name......You can't be me, and if you try, you'll only be a pharisee.I know this because I have tried to seek what had long been denied me.My soul lain open for all to see, and never did YOU look.I thought you might bother to notice, but tis a benighted thought mistook.Thus, a fragmented disposition it holds and it shall so remain.While the world resumes it's orbit, I renew my pain.You can't be me, and if you try, you'll only be a fake.I know this because I've tried, I don't know how much more of this to take.I want to find myself, but there's no one to rely on.No helping hands, no arms, no shoulders to cry upon.So, now I see that all this fruitless labor doneWas only for me to realize that I can trust no one.
