title: "If Only and What If"
writer: "dianes"

If Only I had a mountain filled,
With What Ifs and If Onlys,
I would use some of them for me,
And distribute some to you.

How many times we say,
What If and If Only to ourselves,
Wondering What If and If Only
Would change how we felt.

If Only this wouldn’t be like this,
What If it would be like that,
If Only you would love me,
What If I loved you back?

If Only I wasn’t so fat,
What if I were skinny?
If Only you could know,
What If I knew too.

If Only I knew when I died,
I would go to Heaven,
But What If I go to Hell,
Would it be much different?

Well I don’t have a mountain,
Of If Onlys and What If’s,
If Only I could answer these questions,
IF ONLY, AND WHAT IF!
