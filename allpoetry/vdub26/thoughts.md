I stare off into space
What I think about, I don't know
I just let thoughts run through my head
Let them all pass, pausing briefly on each
All these thoughts, some bad, some good
I think about so much
Life, death, after death
Friends, foes, family, others
So much to think about, so little time
But it keeps me alive
