I really have considered-
Laying down my pen.
And tearing up my manuscripts-
To never write again.

The ink I use is sorrow...
My vellum is my pain...
I really think it better far,
To never write again.

Each word is like a drop of blood-
Each heartbeat hastes the end...
No doubt, there is a better way-
One’s fleeting life to spend...

The tears of memory burn my eyes-
The thoughts, “What might have been”-
You’ll understand? It’s best that I-
Should never write again.
