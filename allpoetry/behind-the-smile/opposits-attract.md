Dark days
Bright nights
So wrong 
Felt right 
Deep scratch
Light cut
Took time 
Hurried up 
White lie 
Dark truth 
Thoughts that hurt
Words that soothe
Imiginary
But so real
Shook hands
Had no deal 
Fixed 
You can still see the cracks 
Has everything 
Something it lacks
Nothing to lose
Everything to gain
A laugh 
Behind smile lies pain
So lost
Seems to be found 
Set free 
Feels bound
Evil or good
Is it that we don't10-19-01,
