There is no point in trying,There is no point in crying.There is no point in staring,There is no point in caring.There is no point in kissing --you left a part of me missing.There is no point in thinking --it only keeps me sinking.There was no point in youThere was no point in meThere was no point in 'us'There was no point in 'we'.You were my other halfleft dieing...Now there is no pointin trying!
