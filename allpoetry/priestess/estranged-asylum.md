...Estranged Asylum...


............................................................



... Estranged Asylum ... 
... languish ... reprise ... 
solidifying vaporous gasps 
... with vibrations of lunacy ... 

... insolent ... sanctions ... 
wardens of reproach 
... saturate surreal delusions ... 
... tentacles of tyranny ... 
        hold me ... 

... Syringed Euphoria ... 
... gripping ... prodding ... 
medicating dreams of escape 
... lying on steel beds ... 

... starving ... injustice ... 
becoming my sanctuary 
... embracing physical intrusions ... 
... a silent soliloquy ... 
        set free ... 





S.S.Snapp
