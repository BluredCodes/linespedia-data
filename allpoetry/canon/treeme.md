The tree that hurts in my backyard
The tree back there thats been so scarred
Leaves have fallen, branches have died
If it had tears it would have cried
In the forest but seems so alone
only surroundings were branches and stones
no softness to be found near the poor tree
Am I describing an oak or could it be me
