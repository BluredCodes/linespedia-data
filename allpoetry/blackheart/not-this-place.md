title: "'Not this place.'"
writer: "blackheart"

Stolen little moments,
As we stand against the rain,
Pressing it with sadness,
Wishing it could stop again.
It never used to be like this,
The sun would shine so bright.
But something flew away,
So far away... this isn't right!

Asking you to love me,
Yet, i know it's nothing now.
The nimble hearts we used to hold,
Are suddenly old, i don't know how!
Save the last kiss...
It may just save us one day.
Hold my hand, we'll run,
I DON'T WANT TO BE IN THIS PLACE!

There has to be a better way,
There has to be a chance!
There has to be something more,
Than this love we've broken all apart.

Rage replaced the love life,
We shared so well each night,
Mournful sigh upon our faces,
This is so bad, this isn't right!
Bleeding but together,
There must be something left,
Please... can we just try,
We have to find a way out of this mess!

Survival of the fittest,
I know that we could make it,
But you don't care do you?!
My heart's about to break!
No longer can i touch you,
Gone is smell and taste,
I want to know the truth right now!
I DON'T WANT TO BE IN THIS PLACE!
