I need inspiration 
I'm stuck for words to write
Wish my mind would help me
But it's feeling rather tight

I need some ideas 
To help me write today
So if you got some
Tell me right away

Think I've covered everything
From here to Timbuktu
So if you got an idea
Send it right on throughgive me some ideas :-)
