If home
Is where the heart is
Then I'm home
With you

If life
Is only lessons
Then I don't think
I've learned

If kisses
Are only nothing
Then I want nothing
From you

If hoping
Is only wasting time
Then I'm wasting time
On you

If love
Is only faking
Then I'm faking
For you

If home
Is where the heart is
Then take me home
With youAhh....you can almost taste the syrup....
