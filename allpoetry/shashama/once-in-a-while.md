Once in a while
I would just like
To be loved
Once in a while
I would love
To be touched
Once in a while
I would kinda like
To be cared for
Once in a while
I want to
Be calm for the day
Once in a while
I'd be happy
If just left alone
Once in a while
I'd like
If someone understood
Once in a while
I wish
That you'd be kind
Once in a while
I really hope
That love's not unrequited
Once in a while
Thats all I askSigh
