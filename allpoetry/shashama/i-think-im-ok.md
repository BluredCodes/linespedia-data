title: "I think I'm ok"
writer: "Shashama"

I think I'm ok
I think I'll be fine
I think that I'll live
Just give me some time

It took me a while
To bounce right back
I think I'm ok
So cut me some slack

It's not like I did it
All by myself
Some people were there
So I did have some help

I would like to thank them
For all that they've done
And just to tell them
That they're number one

So I think that I'll live
I think I'll survive
I think I'm ok now
I just need some time

So please, don't you worry
For I'll get along
You'll know this for sure cause
I'll be singing my song
