---
title: "Single Rose"
writer: "angel of darkness"

A single rose stands alone,
                          Weaving in the wind,
                          So gentle, 
                          So strong,
                          It's beauty is breath taking,
                          I'm afraid to touch it,
                          Because it might prick me.
                          I feel so weak compared to this rose,
                          Watching it stand so strong,
                          Against the winds that are forcing it down.
                          Oh to be that rose,
                          So strong, 
                          So delicate,
                          So beautiful.
