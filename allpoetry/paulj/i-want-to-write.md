title: "I want to write"

I want to write for the Sunday TimesI want to write between the linesI want to write for a magazineI want to write speeches for the queenI want to write a Channel Four playI want to write Thought For The DayI want to write ‘WELCOME’ on police cell doorsI want to write down ten score drawsI want to write a comment on Blair’s successI want to write ‘WHY ARE WE SUPPRESSED?’I want to write ‘FIRST’ on Coulthard’s carI want to write ‘BURST’ on Kylie’s braI want to write love letters to the bankI want to write ‘BEER’ on a septic tankI want to write ‘HELP’ on Hague’s baldheadI want to write my obituary when I’m dead.
