title: "Empty"
writer: "AverageStatistic"

source: "AllPoetry"

I feel so empty.
So useless.
So pointless.
The distance between us is unbearable.
My wounds are still open.
I thought they were healed.
So innocent.
So beautiful.
So sacred.
And I ruined it...
