---
title: "If Only......"
writer: "beautifulworld"
slug: "if-only"
source: "AllPoetry"

My eyes, they close
if only for a second
if only to blink
They fold, crinkle
if only to see the darkness
if only to shed a tear
Bottomless, endless, forever sea
if only to be again washed against the grainy sand
if only to dry and shrivel in the sand
Tossed in with the greens, they shimmer and settle
if only to be shoved into a ravenous mouth
by a feeble hand
Focusing, acute and sharp
if only to see a shattered shard of a child
if only to see everything up close
Cold and frosty, fogged and unclear
if only for a glance out the pane
if only to stare out longingly at the sweet child playing in the yard below
Clear, luminescent
if only to see the world for what it is
and crack the ebony with a ray of light
