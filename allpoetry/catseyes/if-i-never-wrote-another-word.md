title: "If I Never Wrote Another Word"

if I never wrote another word
or heard another song
if I never listened to the wind
and wondered what was wrong 

if I never stumbled through the night
or suffered my fair share
if I never questioned what is right
or followed a distant stare

if I never craved the language
or experienced life’s pain
if I never crossed a lonely bridge
or bathed my skin with rain

if I never took the teardrops
and washed my face with them
if I never heard the footsteps
running constantly within

if I never walked the narrow
or dared to leap the edge
if I never heard the sparrow
singing love songs from the hedge

if I never had a broken heart
or searched so deep within
if I never took the world apart
then put it back again

if I never write another word
if i never pick up the pen
could i fly away, free as a bird
or would i end up nested here

again
