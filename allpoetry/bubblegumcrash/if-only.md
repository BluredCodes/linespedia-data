If Only 


If only you knew how much I love you.
If only you could see the way I watch you.
If only you knew how I’m feeling right now,
When I know you don’t feel the same.

If only you knew how much it hurts,
When I see you with her,
And see how much you love her.
Whilst I’m just a friend.

If only you knew what you mean to me.
If only you’d feel the same.
If only you knew how happy you make me feel,
Just to have you in my life.

You’ll never know any of this.
I’m not going to tell you.
A friend is all I am.
But, if only.ever loved someone from afar?? :(
