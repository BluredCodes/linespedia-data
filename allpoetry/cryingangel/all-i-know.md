---
title: "All I Know"
writer: "cryingangel"

I don't know if I love him.
I don't know if he's true.
All I know is that he always makes me blue.

I don't know if he is serious
I don't know if he's full of sh*t
All I know is that I am always in the pits.

I don't know what my heart says.
I don't know if it calls his name.
All I know is that I think it's all a game.

I don't know if this is real.
I don't know if it's fake.
All I know is this is all a big mistake.

I don't know if he will be there.
I don't know if he will share.
All I know is that he doesn't even care.

I don't know if this is wrong.
I don't know if this is right.
All I know is that we always seem to fight.

I don't know if this feeling will fade.
I don't know if this feeling will dull.
All I know is that you are full of so much bull.Tell me if you like it!!!
