---
title: "If I Knew"
writer: "cryingangel"

If I knew it was a game,
I wouldn't be the same.
If I knew I would be played,
I would never have stayed.

If I knew it was all a joke,
I wouldn't have let you poke.
If I knew you would say goodbye,
I would have asked why.

If I knew it wasn't from the heart,
I would have been very smart.
If I knew you would never be here,
I wouldn't have shed a single tear.

If I knew that you had lied,
I wouldn't have tried.
If I knew you were messing with my head,
I would never have let you get me into bed.

If I knew I would have been deceived,
I never would have believed.
If I knew you never wanted more,
I would have walked out the door.

If I knew...
I would not have been so true.
