If only I hadn't met him
I wouldn't hurt like so
If only I didn't care for him
I wouldn't of had to let him go
If only I hadn't liked him
I wouldn't care that he went away
If only I didn't love him now, 
I wouldn't be crying inside everydayIts short, and to the point, i have mny more, but dont have my book with me...keep checking, there will be many many more
