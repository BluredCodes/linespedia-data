I couldn't begin to count
the times it passes my lips...
one more cut,
one more tear,
one more chocolate,
one more page,
one more minute,
one more pill,
one more smile,
one more frown,
one more dollar,
one more song,
one more dance,
one more picture,
one more street,
one more stop,
one more start,
one more dream,
one more word,
one more kiss, 
one more shot,
one more drop of blood...
So I'll say it one more time-
one more line:
Goodbye.
But I always was a liar.
