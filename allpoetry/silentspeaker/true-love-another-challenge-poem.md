True love,
What is the real meaning?
I ask myself this question all the time.
But never do I get a true answer.
No matter how hard I try, 
I can not seem to dig an answer out of my heart.
My brain chimes in with an answer.
But never do I believe it.
I always seem to have a reason for thinking about it,
And starting to doubt my mind.

True love,
Is only achieved when you have a true friend to share it with.
True love,
Is only believable when your heart feels the same way as your mind.
True love,
Is when both people feel the same way, about each other. 
True love,
Is more then just a feeling, it is a state of being. 
True love,
Is the feeling that if the whole world were to go up in flames, you would die happy.

True love is all those things and more, true love is all you have to dream about.  It is the one dream that all must have, in order to be truly happy in life.
