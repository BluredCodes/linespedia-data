If I told you this...
I'm not sure what you would say.
If you only knew what hell this brings...
If I told you I loved you what would you say
If I told you I've always loved you how would you react
If I took the time to make you feel special about yourself 
Would you do the same for me?
If I mentioned the fact that you love someone else 
Would you fly into a frenzy?
If I told you the truth about me, what would you do?

If I told you what you want to know, would you still be my friend?this is for all of my friends...tell me what you think
