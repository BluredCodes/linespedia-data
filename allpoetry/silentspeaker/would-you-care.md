---
title: "would you care"
writer: "silentspeaker"
slug: "would-you-care"

If I were to die today
Would you care?
If I runaway never to come back,
Would you notice my absence?
If I were to leave town,
Would you notice the silence?
If I were to stop talking to you
Would you be happy? 
If I were to just disappear 
Would any one notice? 

I don't think so,
I really don't think so.
