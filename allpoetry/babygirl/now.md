---
title: "Now"
writer: "babygirl"
slug: "now"
source: "AllPoetry"

We who used to be so in love,
Are now more like enemies.
We who used to be a pair
Are now apart.
We used to be inseparable,
Now there's so much space can't see you.
I used to be the only girl for you,
Now there are more like three.
I should have listened,
I should have gone,
Now instead,
I'm stuck here with a broken heart.
