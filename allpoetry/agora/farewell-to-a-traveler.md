Farewell to a Traveler


Were we to touch
each other again
it would be
the touch
of rain.

If we were to hear
each other once more
we would hear
whispers
from afar.

If we saw
each other
one more time
we would see
pantomime.


Dry roads
noisy roads
beckon us
to different
horizons.
